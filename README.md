
# Simple To-Do List

A **minimal, modern To-Do List web application** built using **HTML, CSS, and JavaScript**. 

## Features

- Add tasks via input field and button or press **Enter**.
- Mark tasks as **completed** by clicking on them.
- **Delete** tasks individually.
- Clean, modern, and responsive UI.
- Fully static, no backend required.

---

## Technologies Used

- **HTML5** – For structure and layout.
- **CSS3** – For styling, modern UI with rounded corners and shadows.
- **JavaScript** – For interactivity and dynamic task management.
